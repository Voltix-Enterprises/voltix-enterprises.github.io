Experimental, legacy, and side projects from the PanelOS ecosystem

PanelOS-x16

PanelOS-x16 is a conceptual 16-bit operating system inspired by early computing platforms. While not part of the core PanelOS roadmap, it exists as a standalone project for retro enthusiasts and low-level systems experimentation.

Learn more about PanelOS-x16

PoP-x86 Subsystem

The PoP-x86 (Panel-on-Panel x86) subsystem is a compatibility layer built into PanelOS-x86_64. It enables native execution of 32-bit PanelOS-x86 applications within the 64-bit environment, preserving legacy support without compromising modern architecture.

Explore PoP-x86 in PanelOS-x86_64
